Introducción al curso:

Explicación del proyecto:

Este proyecto tiene como fin resolver una problemática en específico a través de una estrategia donde llevaremos a cabo para crear un producto o servicio que se invente de manera disruptiva. Tiene como características: la originalidad, especificación, autonomía, investigación y participación.

Y existen 3 rubros importantes que son: tecnológico, ecológico y educacional. En donde un proyecto así se puede llevar a cabo con total éxito, tenemos que identificar que tipo de estos tres proyectos sería el nuestro.

Tipos de Bitácoras y Documentación

Bitácora:

Un cuaderno que incluye hipótesis, ideas, observaciones, datos y obstáculos que suelen pasar durante un proceso de investigación. Suelen utilizarse para procesos educacionales para tener un claro seguimiento de la labor realizada.

Tipos

Blog: Sitio web de uso personal que se actualiza periódicamente y recopila información continuamente de uno o más autores.

Web: El navegador web se comunica con el servidor web mediante el protocolo web o HTTP para entregar la petición deseada.

Redes sociales: Sitios que permiten el intercambio de información entre personas y empresas que comparten relación personal o profesional.

Contenido audiovisual: Es la entrega de producciones audiovisuales informativas que incluye todos los contenidos cinematográficos, televisivos, radiofónicos o multimedia.

Reporte de avance de clase: Informe que se entrega al finalizar una investigación, sirve para presentar las conclusiones de manera formal.

Documentación:

El proceso de información por el cual otorga datos para identificar la técnica instrumental y auxiliar. Los documentos pueden clasificarse en textuales o no textuales.

Notas de clase:

El resumen de los temas desarrollados, sirven de guía en la investigación.

Obstáculos o dificultades:

Obstáculo es lo que dificulta el fin de la acción. La dificultad es referente al problema que sucede cuando queremos algo.

Comentarios y conclusiones:

El comentario es una apreciación a una información analizada y la conclusión es la síntesis de los puntos más importantes de una investigación, solamente de la información previamente dada.

Definiciones y estructura web

Definiciones generales

Sitio web: Web estática: Contenido igual para todos y web dinámica: Contenido cambia según usuario o base de datos.

Página web: Es un documento electrónico accesible desde cualquier navegador web, creado utilizando código HTML.

Dominio: Un dominio es la dirección única que identifica un sitio web en Internet.

Hosting o alojamiento: Espacio físico donde se almacena el sitio web. Ejemplo: webempresa, bluehost, hostgator, punto pe.

Definiciones HTML

Son las siglas de HyperText, markup language o Lenguaje de marcas de hipertexto.

Estructura de HTML

Cabecera es “HEAD”, cuerpo es “BODY”, párrafo es <p>. Las partes principales son: la etiqueta de apertura, la etiqueta de cierre, el contenido y el elemento.

Los atributos contienen información adicional como es class que se utiliza después para apuntarle al elemento de estilo. Debe tener siempre: espacio entre este y el nombre, el nombre seguido por el signo y comillas de apertura y cierre, encerrando el valor del atributo.

Para resaltar paralelas usamos el strong, se usa después de que alineamos correctamente después de la p de párrafo.

Para imagen usamos “img” src = “images/ nombre de la foto.jpg” y procedemos a cerrar. Title es para title, meta charset establece el conjunto de caracteres que debe usar el documento, body es el contenido de la página, listas es li, enlaces usamos “a” y el href atributo al elemento.

Definiciones de CCS

Cascading style sheets. CSS es un lenguaje de hojas de estilo. Ejemplo: p { color: red; }

Selector: para diseñar un elemento diferente cambie el selector. Declaración: especifica cuál de las propiedades del elemento desea diseñar. Propiedades: delice qué propiedades afectar en la regla. Property value: el valor de propiedad permite elegir una de las tantas apariencias, como el color.

JavaScript

Es un lenguaje de secuencias de comandos que se utiliza para agregar interactividad y dinamismo a las páginas web. Permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y realizar una variedad de funciones interactivas en una página web.

Editor

Programa informático que permite crear y modificar archivos digitales. Ejemplo: visual studio code, sublime texto, notepad, etc.

Introducción a sitios web en Google Slides

Introducción a sitios web en Google Slides

Se trata de una introducción al uso de Google Slides para diseñar sitios web. Google Slides es una herramienta de presentación que puede adaptarse para crear sitios web simples.

Diseñar página de inicio

Se refiere a la creación de la página principal de un sitio web. La página de inicio es crucial ya que es lo primero que ven los visitantes. Debe ser atractiva, clara y fácil de navegar.

Personalizar diseños en Google Slides

Describe cómo modificar y personalizar los diseños predefinidos en Google Slides para adaptarlos al diseño deseado del sitio web. Esto puede incluir cambios en colores, fuentes, disposición de elementos, etc.

Menú de investigación

Sugiere la creación de un menú de navegación que facilite la exploración del sitio web. Este menú debe ser intuitivo, claro y proporcionar acceso fácil a todas las secciones importantes del sitio.

Documentación web

Documentación web

Imágenes:

Se refiere a la inclusión de imágenes en una página web. Las imágenes pueden mejorar la estética y la comprensión del contenido.

Enlaces:

Trata sobre la creación de hipervínculos o enlaces que conectan diferentes páginas web entre sí. Los enlaces permiten a los usuarios navegar fácilmente por un sitio web.

Listas ordenadas y no ordenadas:

Las listas ordenadas presentan elementos en secuencia numérica o alfabética.

Las listas no ordenadas presentan elementos sin un orden específico y generalmente se visualizan con viñetas o puntos.

Formato de texto:

Etiquetas:

Se refiere al uso de etiquetas HTML para dar formato al texto en una página web. Las etiquetas pueden aplicar negrita, cursiva, subrayado, tamaño de fuente y otros estilos al texto.
